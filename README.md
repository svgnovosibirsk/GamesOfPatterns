# GamesOfPatterns

In this project each screen implements one of the following patterns:

    # Creational
    * Factory method
    * Abstract factory
    * Builder
    * Prototype
    * Singleton
    
    # Structural
    * Adapter
    * Bridge
    * Composite
    * Decorator
    * Facade
    * Flyweight
    * Proxy
    
    # Behavioral
    * Chain of responsibility
    * Command
    * Iterator
    * Mediator
    * Momento
    * Observer
    * State
    * Strategy
    * Template method
    * Visitor
